Para acessar o Cadastro do Produto é necessário acessar as opções Cadastros > Apoio > Produto conforme imagem abaixo. Através da tela o usuário pode cadastrar um novo Produto, Pesquisar ou Editar produtos já cadastrados.

 

 

Para cadastrar um novo produto basta acionar o botão Novo:

 

O usuario deve preencher todos os campos basicos. Os campos com * são de preenchimento obrigatório.

 

 

  • O campo Tipo Recebimento é carregado de acordo com o Tipo de Material selecionado.É importante ressaltar que a escolha do mesmo irá impactar na executação das tarefas.

Ex: Se o tipo de receipt for por quantidade, nas tarefas o usuario tera de informar e confirmar a quantidade, se for por série o usuario tera que informar os IMEIS nas tarefas.

        

 

 

 

O usuario deve preencher os dados de Dimensões da Caixa, Dimensões da Unidade do Produto, Estocagem, Caixa, Paletização, Auditoria e acionar o botão > Salvar.

 

 

Edição

O upload das fotos do produdo é feita através da edição do mesmo:

 

O sistema apresentara a tela com os dados para a edição. Para adicionar as fotos do produto o usuário deve acionar o botão > Upload Foto 1 – Principal.

 

 

Na modal Upload Foto do Produto acionar o botão > Adicionar.

 

 

Após selecionar uma imagem e fechar a modal, o sistema exibirá a imagem no bloco de fotos.

Obs:. O usuário pode adicionar até três imagens do produto.
