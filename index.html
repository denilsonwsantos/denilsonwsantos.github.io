MANUAL WMS CELÍSTICS


Olá, seja bem-vindo ao Manual WMS Celístics!

Neste manual, você terá todas as informações necessárias para a utilização do sistema WMS em suas operações.

Você conta também com uma equipe especializada para dar todo suporte em suas operações.

Envie seu feedback, dúvida ou sugestão, clicando aqui.

Atenciosamente,

SUPORTE WMS.

VAMOS COMEÇAR?

Fluxo

Produto

Para cadastrar novos produtos, acesse Cadastros > Apoio > Produto , conforme imagem abaixo. Através da tela, você pode Cadastrar um novo Produto, Pesquisar ou Editar produtos já cadastrados.

Para cadastrar um novo produto basta acionar o botão Novo:

O usuário deve preencher todos os campos básicos. Os campos com * são obrigatório.

O campo Tipo Recebimento é carregado de acordo com o Tipo de Material selecionado. É importante ressaltar que a escolha do mesmo irá impactar na executação das tarefas.

Ex: se o Tipo Recebimento for por Quantidade, nas tarefas, o usuário deverá informar e confirmar a quantidade. Se for por Série o usuário deverá informar os IMEIS nas tarefas de separação.

O usuário deve preencher os dados de Dimensões da Caixa, Dimensões da Unidade do Produto, Estocagem, Caixa, Paletização, Auditoria e acionar o botão Salvar.

Armazém

Para acessar o Cadastro de Armazém é necessário acessar as opções Cadastros > Estoque > Armazém conforme imagem abaixo. Através da tela o usuário pode cadastrar um novo Armazém, Pesquisar ou Editar armazéns já cadastrados.

O usuario deve preencher os campos necessarios, e acionar o botão Salvar:

Após o cadastro, é necessario configurar a “Posição Rack”. Para configurar basta pesquisar o armazem, e acionar o botão destacado abaixo:

Nesta etapa o usuario deve preencher todos os campos, pois é através dos mesmos que o sistema montara o layout do armazem:

Ao acionar o botão “Montar Layout”, o sistema monta e exibe o layout do armazem, nesta etapa o usuario deve informar o metro cúbico, e definir quais serão as areas de PICKING.

 

Edição

Acesse a Tela de Edição pesquisando o produto desejado e clicando em ações:


O sistema apresentará a tela com os dados para a edição.

Nessa tela, você pode além de alterar as informações cadastrais do produto, fazer o upload de fotos do mesmo.

Para adicionar as fotos do produto o usuário deve acionar o botão Upload Foto 1 – Principal.

Na modal Upload Foto do Produto clique no botão > Adicionar.

Após selecionar uma imagem e fechar a modal, o sistema exibirá a imagem no bloco de fotos.

Obs:. O usuário pode adicionar até três imagens do produto.

Nota Fiscal de Entrada

Para iniciar o processo de recebimento, é necessário acessar as opções Estoque > Recebimento > Receber Nota Fiscal Entrada conforme imagem abaixo.

No bloco Nota Fiscal de Entrada o usuário deve preencher os dados referente a nota fiscal:

Preencher os dados dos blocos Volumes da Nota Fiscal e Tributos.

No bloco Itens da Nota Fiscal de Entrada, o usuário deve adicionar os produtos presentes na Nota Fiscal.

Para adicionar um produto, o usuário deve buscar pelo código do produto ou descrição, e preencher os demais campos. Caso a nota possua mais de um produto basta acionar o botão Adicionar Item.

Para finalizar o Recebimento de Nota Fiscal de Entrada o usuário deve acionar o botão Receber Produtos.

O sistema apresentará uma mensagem de salvo com sucesso:

Confirmar Recebimento

Após cadastrar o Recebimento de Nota Fiscal de Entrada, o usuário deve confirmar o recebimento dos produtos, para isso basta acessa a tela Estoque > Tarefas

Após preencher os filtros, o usuário deve acionar o botão Pesquisar.

Acionar o link Executar Tarefa na coluna Ação para acessar a confirmação de recebimento.

Se a NF de recebimento possuir mais um produto o sistema gerará uma tarefa para cada produto.

Na tela de Confirmação de Recebimento o usuário deve confirmar o recebimento daquele produto seguindo a configuração do Tipo de Recebimento do mesmo, neste caso o produto possui o tipo Recebimento Por Série Entrada/Saída, logo teremos que informar as Séries(IMEIS) para isto o usuário deve acionar o botão Adicionar Lista de Séries:

Após inserir as Séries (IMEIS) o usuário deve acionar o botão Confirmar.

Divisão de Produtos

Após confirmar o recebimento do produto, é necessário efetuar a divisão do produto. Para isto basta acessar a tela Menu > Estoque > Recebimento > Divisão.

O usuário deve informar o número da Nota Fiscal que deseja efetuar a divisão, e acionar o botão pesquisar:

Acionar o link > Efetuar Divisão na coluna Ação para acessar a tela de Divisão de Produtos.

O usuário deve preencher o Depósito e Local que deseja alocar o produto. Caso seja necessário dividir o produto em mais de um local, basta acionar o botão destacado em verde na imagem abaixo:

Para confirmar as divisões basta acionar o botão Criar Divisão.

Confirmar Divisão

Para confirmar a tarefa de divisão o usuário deve acessar a tela de tarefas: Menu > Estoque > Tarefas

Para consultar a tarefa de divisão da NF o usuário deve selecionar Tipo Divisão, preencher os demais filtros que julgar necessário e acionar o botão Pesquisar.

Acionar o link Executar Tarefa na coluna Ação.

Na tela de Confirmação de Divisão acionar o botão Confirmarpara confirmar a divisão.

1 – Quando há duas divisões o sistema pede confirmação apenas de uma divisão.

Reversa

O processo de reversa ocorre quando o Produto retorna ao Centro de Distribuição.

Para iniciar o processo de recebimento da NF Reversa, é necessário acessar as opções Estoque > Recebimento > Receber Nota Fiscal Entrada conforme imagem abaixo.

Para determinar que a Nota Fiscal é Reversa, é necessárioque o usuário selecione o Check box “Reversa”, a partir desta ação o sistema exibira um novo campo para que o mesmo possa selecionar o Tipo de Reversa.

Neste exemplo iremos selecionar o tipo “Insucesso”, que é quando o produto saiu para entrega, mas não havia ninguém para recebe-lo.

Após selecionado os dois campos o usuario deve preencher os demais campos da NF, e acionar o botão “Receber Produtos”:

Após o processo de Recebimento Reverso, é necessário Confirmar Rebimento da NF Reversa que são as mesmas etapas da Confirmação da Nota Fiscal Recebida. Caso tenha dúvidas quanto aos processos, clique no link abaixo:

  • Confirmar Nota Fiscal Recebida

Na consulta deve-se informar o Armazém, Depósito, e demais campos que o usuario julgar necessario:

O usuario também pode utilizar o botão exibir pra verificar em qual posição Rack do Depósito está este produto:

Cadastrar Pedido de Venda

Para acessar a tela de Cadastro de Pedido de Venda o usuário deve acessar Comercial > Pedido de Venda, através desta tela o mesmo pode pesquisar ou cadastrar um novo pedido:

Para cadastrar um novo pedido deve-se acionar o botão Novo:

No bloco Dados do Pedido de Venda o usuário deve preencher os dados referente ao Pedido:

No bloco Cliente para Entrega o usuario deve selecionar o Cliente (o mesmo que foi configurado na filial) para o qual sera entregue o produto:

Apartir da seleção do Cliente o sistema abrira um Modal, onde o usuario deve selecionar o endereço de entrega do Cliente:

No bloco Itens de Pedido o usuario deve selecionar o Armazém e o depósito a qual se encontra o(s) produto(s) requerido(s) no pedido.

O usuario deve preencher neste mesmo bloco o produto e quantidade requerida do mesmo, e acionar o botão “Montar Item”. Neste momento o sistema valida se aquela quantidade existe no depósito e se está em uma área de picking, caso exista e esteja ele monta o item.

Após a montagem o sistema permite ao usuario através do botão “Exibir posições utilizadas”, visualizar as posições do rack de onde seram retirados os itens:

Gerar Picking List

Antes de separar o produto no depósito, é necessário imprimir ou gerar o Picking list. Para isto, basta acessar a tela Estoque > Geração Picking List:

O usuário deve selecionar o Armazém, Depósito, tipo de impressão e demais campos que julgar necessário, feito isso deve acionar o botão “Pesquisar”:

O mesmo deve selecionar o Pedido, e acionar o botão Imprimir Picking:

Separar Pedido por Fornecimento

Após imprimir o picking list é necessario que o usuario separe o pedido por fornecimento, para isto basta que o mesmo acesse Estoque à Separar Pedido por Fornecimento.

O usuario deve informar o numero do fornecimento, e acionar o botão Pesquisar:

Na tela de Separação dos Produtos o usuario deve preencher o pallet e o volume,e acionar o botão Confirmar:

Após confirmar a separação, o sistema exibe a mensagem de sucesso:

Executar Tarefa de Separação

Após separar o pedido no estoque é o momento de separar o produto no depósito, para exucutar esta tarefa o usuario deve acessar Estoque > Tarefas :

O usuario deve preencher os campos que julgar necessario, e acionar o botão “Pesquisar”:

Após a pesquisa, o mesmo deve acionar o link Executar Tarefa:

Na separação é necessario que o usuario informe o Pallet, volume do produto e adiocione as séries(IMEIS) dos mesmos:

Após realizar a leitura de todas as Séries, o usuario deve acionar botão Separar, neste momento o sistema valida as Séries lidas, e se as mesmas estão no depósito correspondente ao selecionado no pedido.

Cadastrar Nota Fiscal

É importante que antes de importar a Nota Fiscal de saída o usuário valide se o pedido está pronto para faturamento, para isto o mesmo deve acessar Estoque > Saída > Pedidos Prontos para Faturamento.

Conforme podemos ver na imagem abaixo o pedido já está com status pronto para faturar, logo o usuário pode prosseguir com a importação da nota fiscal de saída.

Quando o sistema carregar o Pallet no grid, basta o usuario acionar o botão Importar Notas Fiscais de Saida:

Na tela de importação existe um arquivo de exemplo a qual auxiliara o usuário no preenchimento dos dados a serem importados.

Após preencher o arquivo, é necessario que o usuario importe o mesmo no sistema, para isto o mesmo deve selecionar o arquivo através do botão “Escolher arquivo...” e após isto acionar o botão Importar:

Após concluir a importação, o sistema exibe as Notas Fiscais que foram importadas com sucesso ou insucesso:

Transferência de Material

Este cenário ocorre quando é necessário movimentar o produto dentro do estoque, seja pra uma área de picking ou não. Para efetuar tal procedimento o usuário precisa acessar a tela Estoque > Pedidos de Movimentação.

Acionar o botão Novo:

No pedido de movimentação o usuário deve selecionar o Tipo = Transferência, informar a Data e Produto, e acionar o botão Pesquisar.

Neste momento o sistema valida se existe aquele produto em estoque, caso exista o sistema exibe o mesmo em tela com sua respectiva quantidade no estoque.

O usuário deve selecionar o Depósito e Rack Destino para o qual ele deseja movimentar o produto, após isso deve acionar o botão Movimentar:

Quando concluido, o sistema apresenta uma mensagem de sucesso em tela:

Executar Tarefa de Transferência

Para confirmar a tarefa de transferência o usuário deve acessar Menu > Estoque > Tarefas:

Para consultar a tarefa de transferência de produto, o usuário deve selecionar Tipo: Transferência, preencher os demais filtros que julgar necessário e acionar o botão Pesquisar.

Acionar o link Executar Tarefa na coluna Ação.

Para confirmar a transferência, basta o usuário acionar o botão Transferir:

Consultar Estoque

Após a transferência o usuário pode validar se a mesma foi efetuada com sucesso através da tela “Situação de Estoque”.

Na pesquisa o usuário deve selecionar o armazém e depósito, e acionar o botão Pesquisar:

Como podemos confirmar na imagem abaixo, a tranferência foi realizada com sucesso para a posição rack A020101:

Subcontratação

Este processo ocorre quando já existente um produto no CD e é necessário fazer uma alteração, classificando parte desses produtos em estoque com um novo nome/produto.

Para efetuar tal procedimento o usuário precisa acessar Estoque > Pedidos de Movimentação:

Acionar o botão Novo :

No cadastro de pedido de movimentação o usuário deve selecionar o Tipo = SUBCONTRATACAO, informar a Data, Produto, e acionar o botão Pesquisar.

Neste momento o sistema valida se existe aquele produto em estoque, caso exista o sistema exibe o mesmo em tela com sua respectiva quantidade no estoque.

O usuário deve selecionar o produto, e informar a quantidade que irá subcontratar, feito isso o mesmo deve acionar o botão “Movimentar”:

Executar Tarefa de Subcontratação

Para confirmar a tarefa de subcontratação o usuário deve acessar Menu > Estoque > Tarefas:

Para consultar a tarefa de subcontratação do produto, o usuário deve selecionar Tipo: subcontratação, preencher os demais filtros que julgar necessário e acionar o botão > Pesquisar.

Acionar o link > Executar Tarefa na coluna Ação.

O usuário deve informar o Novo Produto, e acionar o botão Efetuar Subcontratação:

Consultar Estoque

Após a Subcontratação o usuário pode validar se a mesma foi efetuada com sucesso através da tela “Situação de Estoque”.

O usuario deve preencher o Armazém, Depósito e demais filtros que julgar necessario e após isto acionar o botão Pesquisar:

Como podemos confirmar o produto que foi subcontratado já consta no estoque.
